# 色彩与明暗基础

## 明暗与层次
- 曝光：整体明度的基础，决定高光与阴影是否保留细节。
- 对比：控制层次分离度，适当对比让画面更“立体”。
- 黑场与白场：通过“黑位/白位”塑造画面厚度与干净度。

## 色温与色调
- 色温（温暖/冷峻）：影响氛围与时间感。
- 色调（偏品红/偏绿）：影响肤色与中性灰的稳定。

## 色彩关系
- 主色相：画面最显著的颜色，用于建立风格。
- 对比色：营造张力与视觉冲击（如蓝-橙）。
- 邻近色：营造统一与安静的氛围。

## 质感提升的关键
- 适度降低整体饱和度，提高“明度对比”。
- 保留高光细节，避免死白；阴影可略压但不“糊”。
- 肤色优先，确保肤色处在自然区间。
